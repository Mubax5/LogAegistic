# PRD Delta — 3-Agent Alignment Patch

This patch aligns the MVP with the original 3-core-agent concept.

## Correction

The MVP uses 3 visible core agents:

1. Manifest Tech-Deconstructor
2. Geopolitical Sanctions Bloodhound
3. Logistics Legal-Arbitrator

The previously proposed Adversarial Evasion Analyst is not shown as a fourth core agent in the MVP. Its value is absorbed into the cross-check behavior of the Legal-Arbitrator and the route/ownership review behavior of the Bloodhound.

## Feature Additions

The MVP now emphasizes:

- Case overview for non-technical users
- Clear warehouse release status
- Three-agent tribunal state
- Evidence Packet Board
- Agent Cross-Check transcript
- Decision Matrix
- Decision Defensibility explanation
- Cure Pack with next actions
- Tamper-evident audit trail
- Human review handoff

## Updated MVP Scope Line

Three specialized agents collaborate through Band, using structured evidence packets, cross-check challenges, and legal arbitration to produce a GO/HOLD/NO-GO recommendation.
