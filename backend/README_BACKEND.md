# FastAPI Backend

Run:

pip install -r requirements.txt
uvicorn app:app --reload

Open:

http://127.0.0.1:8000/docs

Main endpoints:

- GET /api/order
- POST /api/tribunal/start
- GET /api/tribunal/evidence
- GET /api/tribunal/transcript
- GET /api/tribunal/decision
- GET /api/tribunal/audit-trail
